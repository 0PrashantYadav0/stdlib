## Usage

```text
Usage: make <command> [<ENV_VAR>=<value> <ENV_VAR>=<value> ...]
```

### Commands

#### list-contributors

Lists contributors to the project repository.

```bash
$ make list-contributors
```

#### update-contributors

Updates the project `CONTRIBUTORS` file.

<!-- run-disable -->

```bash
$ make update-contributors
```
