## Usage

```javascript
var build = require( 'path/to/tools/build-docs' );
```

#### build( done )

Builds documentation.

```javascript
function done( error ) {
    if ( error ) {
        throw error;
    }
    console.log( 'Success!' );
}

build( done );
```
